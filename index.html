ИИ развивается экспоненциально — важно начинать сейчас

  • Понимание «экспоненты» и ускорения прогресса (00:08:13)
  • Идея: в ближайшее время модели начнут удивлять (00:09:40)

Промпты ≠ магия: важнее контекст и постановка задачи

  • Контекст-инжиниринг важнее prompt-инжиниринга (01:27:22)
  • Контекст — ограниченный ресурс, чем больше — тем хуже качество (01:28:05)
  • Объясняйте «как человеку», без ролевых клише (01:28:21)

Работа через код: ИИ должен проверять себя и данные

  • Промпты не важны: дайте задачу и попросите работать кодом (00:54:45)
  • Просите тесты/проверки, чтобы инфографика точно соответствовала данным (00:55:20)

Код-агенты и безопасность: начинайте с «песочницы»

  • Code-Work: изолированная среда без доступа к паролям/файлам (00:51:11)
  • Почему это безопасно для нетехнических людей (00:51:43)
  • Полный доступ возможен, но есть риски безопасности (00:49:48)

Подписка vs API: что выбирать

  • Если работаете на своём компьютере — выбирайте подписку (00:36:36)
  • API — для ботов/сервисов на серверах, но дороже (00:36:46)

Экосистема важнее отдельных моделей

  • Сильные модели vs экосистемы (пример: kimi) (00:34:36)
  • Agent Swarm и «рой агентов» (00:35:05)

Локальные модели: ограничения железа

  • На обычном ПК качество топ-моделей не получить (01:12:08)
  • Для сильных локальных запусков нужны дорогие машины (01:12:54)

Практический совет новичкам

  • Просите пошаговые инструкции под вашу систему/задачу (01:33:34)
  • Старт в VS Code через расширения (дружелюбно для новичков) (01:33:56)

Как пользоваться

  1. Откройте index.html и выберите интересующую идею.
  2. Ссылка перенесёт на нужный таймкод в transcript.html.
  3. В расшифровке доступен поиск по словам (вверху страницы).
